Плагин для работы с массивами

## Использование
```
#beginScript
|DV|[list] = fghfgh
fghfghfgh
fghfghj6j
fghss55s
aspper
rtyhh
jjhjmnh"m
#endScript

|DV|[array] = (|DLL|dllName:array;funcName:from_list;params:|DV|[list];|DLL|)
... обрабатываете массив
|DV|[list] = (|DLL|dllName:array;funcName:to_list;params:|DV|[array];|DLL|)
обычный список
```
